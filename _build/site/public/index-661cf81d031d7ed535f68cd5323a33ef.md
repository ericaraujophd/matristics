# Replicate This Study

All materials needed to replicate this evaluation are published openly. This page describes what you need, what you can reuse as-is, and where adaptation may be required.

## What You Need

1. **API access** to the models you want to evaluate. See {doc}`../data-collection/index` for setup.
2. **Python 3.10+** and the packages listed in {doc}`../data-collection/setup`.
3. **Domain-expert annotators** — at minimum two with graduate-level knowledge of early Christianity. See {doc}`../methodology/annotation-protocol` for annotator requirements and training materials.
4. **The rubric** — download from {doc}`../methodology/rubric`.
5. **The question bank** — download from {doc}`../figures/index`.

## What You Can Reuse Without Modification

- The full question bank (42 questions, {download}`CSV <../figures/question_bank.csv>`)
- The rubric definitions and scoring sheet template ({doc}`../methodology/rubric`)
- All data collection notebooks ({doc}`../data-collection/index`)
- The annotation protocol ({doc}`../methodology/annotation-protocol`)
- The reference answer document *(to be published when annotation is complete)*

## What May Require Adaptation

- **New models**: add a new notebook following the pattern of the existing API tutorials. The shared system prompt and data format should be used unchanged for comparability.
- **New figures**: add a new page to the Figures section following the template of {doc}`../figures/macrina`. Add three questions (Basic / Intermediate / Advanced) to the question bank CSV.
- **Different annotators**: re-run the pilot round with your annotators to establish local calibration. Report your inter-annotator alpha alongside any results.
