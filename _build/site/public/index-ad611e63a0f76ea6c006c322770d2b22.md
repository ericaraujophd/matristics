# Benchmark

This section presents the evaluation results as a comparative panel. Results are organized across three views: by model, by rubric dimension, and by figure. Together, they allow readers to see not only which model performed best overall, but where each model's weaknesses lie and which historical figures are most poorly represented.

```{note}
Data collection and annotation are in progress. Results will populate automatically as annotation is completed.
```

## Views

- {doc}`by-model` — Overall model ranking and per-question score distributions
- {doc}`by-dimension` — Which models hallucinate least? Which have the worst gender bias scores?
- {doc}`by-figure` — Which figures are most poorly represented across all models?

## How to Read the Results

Each LLM response is scored on seven dimensions (1–5 scale):

| Dimension | What a high score means |
|-----------|------------------------|
| Factual Accuracy | Correct historical claims |
| Completeness | All major aspects addressed |
| Gender Bias | Female figure presented as agent in her own right |
| Hallucination | No invented content |
| Anachronism | No projection of later frameworks |
| Source Awareness | Sources correctly distinguished |
| Depth & Nuance | Scholarly debate acknowledged |

Average scores are reported per model, per dimension, and per figure. Krippendorff's alpha inter-annotator agreement statistics are reported alongside all scores.
